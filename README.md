# Full Stack Open - Phonebook App

This repository contains my solutions for the Full Stack Open course.

## 📱 Phonebook Application

A fullstack phonebook app built with:

* React (Frontend)
* Node.js & Express (Backend)

## 🌐 Live Application

👉 👉 [Live App](https://fullstackopen-3q17.onrender.com/)

## 🛠 Features

* Add new contacts
* Delete contacts
* Filter/search contacts
* Backend API integration

## 📂 Project Structure

* `part3/phonebook` → Backend + production frontend build
* `part2` → Frontend development version

## 🚀 Deployment

The app is deployed using Render.
